# Shopflow
Plateforme SaaS de gestion commerciale multi-boutiques.

## Technologies
- Next.js (App Router)
- Tailwind CSS
- Shadcn UI
- Framer Motion
- Recharts
- Next Themes

## Lancement

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
